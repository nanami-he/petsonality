## 2. Reddit r/ClaudeAI

**Title:** I loved Claude's buddy dragon, so I made a full MBTI pet system for your terminal

I kept thinking about Claude's buddy concept — a little creature that watches you code. But I wanted one that actually has personality.

So I built **petsonality**: 16 ASCII animals mapped to MBTI types. Each one speaks differently — the Fox (ENTP) picks apart your logic, the Bear (ENTJ) just says "Fix it", the Parrot (ESFP) repeats what you said but louder.

It lives in your status line, reacts to errors, celebrates when tests pass, and won't stay silent if you've been quiet too long.

- 1276 unique reactions (638 EN + 638 ZH)
- One command: `npx petsonality`
- Works with Claude Code + OpenClaw
- MIT, open source

GitHub: https://github.com/nanami-he/petsonality

Would love feedback — especially on which animal you'd pick.
